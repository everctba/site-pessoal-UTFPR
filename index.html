Meu Behance

Durante o projeto, implementei diversos efeitos visuais e animações, explorando técnicas avançadas de CSS com variáveis para animar ângulos. Também descobri o uso de imagens animadas no formato .avif, otimizando desempenho e qualidade. O site foi desenvolvido em WordPress utilizando o Oxygen Builder.

Este case foi completo, logomarca e identidade visual até o layout do site, passando por posts para redes sociais e embalagens. Usei Blender 3D para renderizar packshots publicitários, levando em consideração soluções viáveis e com ótimo custo-benefício, alinhando estética e conceito de marca.

Nesse projeto, tive a oportunidade de aprender e implementar animações com LottieFiles. Desenvolvi o código de uma página enquanto aprendia o básico do Nuxt, framework baseado em Vue, e comecei a trabalhar com Tailwind CSS.
